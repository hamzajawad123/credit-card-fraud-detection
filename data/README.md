# Data Folder

Place the dataset file here before running the notebook:

- `credit_card_fraud_10k.csv` — 10,000 credit card transactions with fraud labels

## Dataset Details

| Property | Value |
|---|---|
| Rows | 10,000 |
| Columns | 10 |
| Fraud rate | 1.51% |
| Target column | `is_fraud` |

## Note

The dataset file is included in this repository for reproducibility.  
If you are cloning this repo, the CSV file should already be present in this folder.

If running in **Google Colab**, upload this CSV file via the Colab sidebar Files panel.
